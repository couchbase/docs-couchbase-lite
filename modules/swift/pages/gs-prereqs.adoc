.Vector Search Prerequisites
[IMPORTANT]
--
Vector Search is only available for 64-bit architectures and
Intel processors that support the Advanced Vector Extensions 2 (AVX2) instruction set.
Most machines manufactured within the last ten years support this instruction set.
You can check if your machine supports the AVX2 instruction set by following the instructions https://www.intel.com/content/www/us/en/support/articles/000090473/processors/intel-core-processors.html[here.] 
--
