== Get Started

To get started coding Couchbase Lite for Swift apps, just create or open an existing _Xcode_ project and install Create or open an existing _Xcode_ project and install _Couchbase Lite {this-release}_ using one of the <<lbl-install-tabs>> methods shown.

[NOTE]
--
Couchbase Lite is distributed as an xcframework, to support this:

* The minimum required Cocoapods version is *1.9*
* The minimum required Carthage version is *38.0*
--
